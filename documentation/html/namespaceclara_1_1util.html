Iterates over all iterators with a function that appends its result to OutputIt Iterators have to have the same size! Example:

std::vector<int> xs { 1, 2, 3 };
std::vector<int> ys { 2, 4, 8 };
std::vector<int> res { 0, 0, 0 };
zipWith([](int x, int y){ return x+y; }),
xs.begin(),
xs.end(),
res.begin(),
ys.begin());
// res = { 3, 6, 11 };
